Effect of Interface Closeness on Abstract Problem Solving
Thesis. This work investigated the effects of interface closeness on abstract problem solving. Interface closeness is the degree to which an action performed via the interface differs from the action performed to achieve the desired result in the natural world. It has been suggested that closer interfaces, such as touch and tangible, would yield poorer performance on problem solving tasks as a result of their more natural interaction style. However, no empirical investigations have been conducted to test this assumption.

Perception of Cube Rotations
As part of a joint research project between the departments of Computer Science and Psychology, I developed software to replicate and further explore Shiffrar & Shepard (1991).

Tangible Password
TangibleRubik, a novel tangible password system, allowed users to physically manipulate a Rubik's Cube to enter a password. By having users physically manipulate the cube, the system took advantage of humans' innate ability to recall motor actions through repetition.

World of Warcraft Accessibility Project
Many visually intensive applications, such as World of Warcraft, are disenfranchisig to users with low visual acuity. With a universal access goal in mind, we created and tested an enhanced user interface for the game. Results showed that even minor improvements led to significant benefits.

Using Connectionist Models to Classify Sounds
This project utilized connectionist networks (or artificial neural networks) to learn how to classify sounds (i.e. auditory tones) into dintinct categories based on explicit or implicit rules. The goal of the project was to model human sound classifcation data with the our network model.
